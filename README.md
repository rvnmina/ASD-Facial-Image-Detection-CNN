## Project Overview
This project explores automated Autism Spectrum Disorder (ASD) detection
using facial image analysis and Convolutional Neural Networks (CNNs).
To improve interpretability, SmoothGrad-CAM++ is used to visualize
important facial regions influencing predictions.

## Files
- `code/visual_project.ipynb` – Model training, evaluation, CAM++
- `report/visualproject.pdf` – Detailed project report
- `presentation/visualpresentation.pdf` – Final presentation slides

## Dataset
Kaggle Autism Children Face Image Dataset

## Methods
- Custom CNN (PyTorch)
- Adam optimizer
- Cross-entropy loss
- SmoothGrad-CAM++ for interpretability

## Results
- Test Accuracy: ~79%
- Balanced precision/recall across classes

## Disclaimer
This project is for academic and educational purposes only.
